FiftyOne.IpIntelligence.Engine.OnPremise.FlowElements.IpiOnPremiseEngine Class Reference

Detailed Description

IP intelligence engine.

This engine takes IP addresses and other relevant HTTP headers and returns properties about the IP range that the IP address fall within
